Name: symbolconfiguration\generate_address_info

Required Runtime Components: CmpIecVarAccess

Default Value: False

Recommended Value: False

XML Type: boolean

dotNet Type: System.Boolean

Example:

<ts:section name="symbolconfiguration">
      ...
    <ts:setting name="generate_address_info" type="boolean" access="visible">
        <ts:value>0</ts:value>
    </ts:setting>
      ...
</ts:section>

Documentation:

Include the address information in the generated XML file. This can be
used to implement symbolic access without the IecVarAccess component
and library, for example directly by a different runtime component, or
via the monitoring protocol (PLCHandler library). Useful for small
PLCs which don't support OPC or even Symbol Configuration.

PlugIn:

SymbolConfigObject.plugin (3.5.11.0)
