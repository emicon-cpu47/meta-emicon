transfer_xml_onlyÂ¶

Name: symbolconfiguration\transfer_xml_only

Required Runtime Components: CmpIecVarAccess

Default Value: False

Recommended Value: False

XML Type: boolean

dotNet Type: System.Boolean

Example:

<ts:section name="symbolconfiguration">
      ...
    <ts:setting name="transfer_xml_only" type="boolean" access="visible">
        <ts:value>0</ts:value>
    </ts:setting>
      ...
</ts:section>

Documentation:

Only the XML will be generated, but no binary symbol information or
code is actually generated nor downloaded to the device. This is
useful for very small devices, or when the symbol configuration logic
is on a different device. Usually is combined with the
'symbolconfiguration\generate_address_info' setting.

PlugIn:

SymbolConfigObject.plugin (3.5.11.0)
