Name: runtime_features\disable_device_application

Default Value: False

Recommended Value: False

XML Type: boolean

dotNet Type: System.Boolean

Example:

<ts:section name="runtime_features">
      ...
    <ts:setting name="disable_device_application" type="boolean" access="visible">
        <ts:value>0</ts:value>
    </ts:setting>
      ...

Documentation:

Type: boolean; TRUE or 1: the device does not support the DeviceApplication structure.
                  False or 0: default behavior, the user decides whether the new DeviceApplication structure
                  or the existing application structure is used in the project.
