Documentation:

A list of visual elements which are supported by the device. If this list is empty all elements of the active visualization profile
are available. Only one of the both targetsettings 'AvailableElements' and 'NotAvailableElements' should be used to restrict the number of available elements. 
The control NativeControl has had another behaviour until V3.5.6. If only these special control is in the list also all other controls of the active visualization 
profile are available. For additional controls which must be activated by the target use the target setting AdditionalAvailableElements.
