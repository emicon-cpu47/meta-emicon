additionalkeys¶

Name: visualization\keyboardusage\additionalkeys

XML Type: string

dotNet Type: System.String

Example:

<ts:section name="visualization">
      ...
    <ts:section name="keyboardusage">
          ...
        <ts:setting name="additionalkeys" type="string" access="visible">
            <ts:value></ts:value>
        </ts:setting>
          ...
    </ts:section>
      ...
</ts:section>

Documentation:

This section allows the definition of additional keys that should be available in the visualization.
Please see the following example for an explanation of the definition additional keys (two keys EMERGENCY_STOP, MACHINE_SHUTDOWN):
	<ts:section name="additionalkeys">
		<ts:setting name="EMERGENCY_STOP" type="integer" access="visible">
		<ts:value>300</ts:value>
		</ts:setting>
		<ts:setting name="MACHINE_SHUTDOWN" type="integer" access="visible">
		<ts:value>301</ts:value>
		</ts:setting>
	</ts:section>
The names of these keys shown to the user can be localized using the according mechanisms of device descriptions. 
Please see the following example:
	<Strings namespace="keyboardusage">
		<Language lang="de">"
		<String identifier="EMERGENCY_STOP">
			NOT AUS
		</String>
		<String identifier="MACHINE_SHUTDOWN">
			Maschine ausschalten
		</String>
		</Language>
		<Language lang=“fr“>“
		<String identifier="EMERGENCY_STOP">
			Arrêt d’urgence
		</String>
		<String identifier="MACHINE_SHUTDOWN">
			Eteindre la machine
		</String>
		</Language>
		<Language lang="en">"
		<String identifier="EMERGENCY_STOP">
			Emergency Stop)
		</String>
		<String identifier="MACHINE_SHUTDOWN">
			Shut down machine
		</String>
		</Language>
	</Strings>
