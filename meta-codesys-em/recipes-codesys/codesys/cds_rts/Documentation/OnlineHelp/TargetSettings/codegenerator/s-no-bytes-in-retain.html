Name: codegenerator\no-bytes-in-retain

Default Value: False

Recommended Value: False

XML Type: boolean

dotNet Type: System.Boolean

Example:

<ts:section name="codegenerator">
      ...
    <ts:setting name="no-bytes-in-retain" type="boolean" access="visible">
        <ts:value>0</ts:value>
    </ts:setting>
      ...

Documentation:

Special setting for special hardware: if bytes cannot be addressed in
retain memory, then all 1 Byte size data types in Retain memory will
allocate 2 Bytes.
