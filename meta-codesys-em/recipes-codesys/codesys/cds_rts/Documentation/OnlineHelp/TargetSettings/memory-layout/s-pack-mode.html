Name: memory-layout\pack-mode

Default Value: 8

Recommended Value: 8

XML Type: integer

dotNet Type: System.Int32

Example:

<ts:section name="memory-layout">
      ...
    <ts:setting name="pack-mode" type="integer" access="visible">
        <ts:value>8</ts:value>
    </ts:setting>
      ...
</ts:section>

Documentation:

The pack-mode.
4= structures are created 4 - byte - aligned, i.e.: 
BYTE on BYTE limits 
WORD on WORD limits 
DWORD on DWORD limits 
LINT on DWORD limits 
8= structures are created 8 - byte - aligned, i.e.: 
BYTE on BYTE limits 
WORD on WORD limits 
DWORD on DWORD limits 
LINT on LINT limits  
[Default=8]  
Within a structure, bytes are aligned to byte limits, words to word 
limits etc. 
On the other hand a variable of type of a structure will be aligned to 
the limit of the biggest non - structured data type which is contained.
