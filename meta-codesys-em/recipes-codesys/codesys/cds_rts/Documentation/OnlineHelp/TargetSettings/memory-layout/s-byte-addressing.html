Name: memory-layout\byte-addressing

Default Value: False

Recommended Value: False

XML Type: boolean

dotNet Type: System.Boolean

Example:

<ts:section name="memory-layout">
      ...
    <ts:setting name="byte-addressing" type="boolean" access="visible">
        <ts:value>0</ts:value>
    </ts:setting>
      ...
</ts:section>

Documentation:

determines whether the direct addresses with % is addressing bytes or
words in the memory. E.g with this setting true, the address %MW10 is
at byte offset 10 in memory. With this setting false, the address is
at byte offset 20 in memory (default, and recommended).
