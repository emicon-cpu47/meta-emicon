Target VisualizationÂ¶

The Target Visualization feature is used to display a visualization created in CODESYS on a display connected with the controller or directly built in the controller.

The following target settings are necessary for this feature:

  • targetvisualization
  • targetvisualization_insertbydefault

The following runtime system components are necessary for this feature:

  • CmpBitmapPool
  • CmpDynamicText
  • CmpTargetVisu
  • CmpVisuHandler
  • CmpVisuServer
  • SysGraphic
  • SysWindow

The following runtime system interfaces are implemented by the components for this feature:

  • CmpVisuHandlerItf
  • CmpVisuServerItf
  • CmpTargetVisuItf
  • CmpDynamicTextItf
  • CmpBitmapPoolItf
  • SysGraphicItf
  • SysWindowItf
