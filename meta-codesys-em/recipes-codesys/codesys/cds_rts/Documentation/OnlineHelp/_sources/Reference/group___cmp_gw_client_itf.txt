.. _group___cmp_gw_client_itf_1ga754eec70de74ef167ef68fc8b0409ec7:

int GWClientRegisterCommDrv (COMMDRVITF *pItf, COMMDRVINFO *pDrvInfo, RTS_HANDLE *phDriverHandle)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Register a commdriver with the gateway client.

**Parameters**

+---------+-----------------+---------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | pItf            | The interface functions of the comm driver.                                                                                                 |
+---------+-----------------+---------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | pDrvInfo        | Information about the driver. The hDriver field is unused in this call.                                                                     |
+---------+-----------------+---------------------------------------------------------------------------------------------------------------------------------------------+
| [out]   | pulDriverHandle | Will be set with the handle assigned to the communication-driver. The driver must use this handle in all callbacks (eg. GWClientSendReady). |
+---------+-----------------+---------------------------------------------------------------------------------------------------------------------------------------------+

.. _group___cmp_gw_client_itf_1ga23a4c99c20f320b3e33f13d6c713a385:

int GWClientConnectionReady (RTS_HANDLE hDriverHandle, RTS_HANDLE hConnHandle, int nAction)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

May be called by a commdriver if a connection is ready to send/receive. This function shall not signal a condition more then once without the gateway client reacting to that signal. Eg. if the commdriver signals that a connection is ready to send it shall not signal this situation again before the gateway client actually called "send". The gwclient must not rely on this function to be called but may use it to increase the performance of the connection.

**Parameters**

+---------+----------------+-----------------------------------------------------------------------------------------------+
| [in]    | dwDriverHandle | The driver handle returned by GWClientRegisterCommDrv.                                        |
+---------+----------------+-----------------------------------------------------------------------------------------------+
| [in]    | dwConnHandle   | The handle to the connection.                                                                 |
+---------+----------------+-----------------------------------------------------------------------------------------------+
| [in]    | nAction        | The action that is ready on this connection - eg. COMMDRV_ACTION_SEND, COMMDRV_ACTION_RECEIVE |
+---------+----------------+-----------------------------------------------------------------------------------------------+
