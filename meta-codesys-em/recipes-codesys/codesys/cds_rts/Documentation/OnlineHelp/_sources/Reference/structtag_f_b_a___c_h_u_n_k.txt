Detailed Description
--------------------

Memory chunk for managing allocations and de-allocations.

Memory blocks are obtained from the area marked with the unallocated_block index (called 'right-hand area') until at least one block gets returned by free(). Once it happens, the consecutive allocations are performed from the 'left-hand area' containing returned blocks.

Once the left-hand area is exhausted, allocations continue to be made from the right-hand area.
