#define GATEWAYKEY_INT_INACTIVITY_TIMEOUT “InactivityTimeoutMs”¶

Setting to modify the client inactivity timeout in ms. If no piece of data (also no keepalive) is received or successfully sent wihtin this time, the Gateway closes the connection to the client.
