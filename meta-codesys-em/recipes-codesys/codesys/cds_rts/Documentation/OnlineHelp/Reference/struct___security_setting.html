Detailed Description¶

This structure stores a single selection of a selectable settings. Each possible selection is a single struture element when registering the setting. The registration can be done using SecManRegisterSettings function. The selected setting can be retrieved by using SecManGetSetting.

Field Documentation¶

RTS_I32 _SecuritySetting::secID¶
