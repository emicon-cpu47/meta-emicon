Detailed Description¶

Interface of CmpIoDrvC.

This is the interface to get access from an IEC program to every IO-driver written in C. All interfaces of an IO-driver in C must be specified in this wrapper! In the IEC program, the CmpIoDrvC.library is needed to enable this access.
