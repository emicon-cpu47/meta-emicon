#define EVT_CmpMonitor2_PrepareWrite MAKE_EVENTID (EVTCLASS_INFO , 1)¶

Event is sent to prepare the write operation to an expression. Note: this event is compatible with the event EVT_PrepareWriteVariable from the component CmpMonitor.

Parameters

[in] pEventParam EVTPARAM_CmpMonitor2Write

#define EVT_CmpMonitor2_WriteDone MAKE_EVENTID (EVTCLASS_INFO , 2)¶

Event is sent after the write operation to an expression was done. Note: this event is compatible with the event EVT_WriteVariableDone from the component CmpMonitor

Parameters

[in] pEventParam EVTPARAM_CmpMonitor2Write

#define EVT_CmpMonitor2_PrepareForce MAKE_EVENTID (EVTCLASS_INFO , 3)¶

Event is sent to prepare the force of an expression Note: this event is compatible with the event EVT_PrepareForceVariable from the component CmpMonitor

Parameters

[in] pEventParam EVTPARAM_CmpMonitor2Force

#define EVT_CmpMonitor2_ForceDone MAKE_EVENTID (EVTCLASS_INFO , 4)¶

Event is sent after the force operation of an expression was done. Note: this event is compatible with the event EVT_ForceVariableDone from the component CmpMonitor

Parameters

[in] pEventParam EVTPARAM_CmpMonitor2Force
