Macro Definition Documentation¶

#define BLKDRVCANCLIENTKEY_INT_TASKPRIO “TaskPrio”¶

Task priority. Default: 95 (HighEnd)

#define BLKDRVCANCLIENTKEY_INT_TASKPRIO_DEFAULT RTS_TASKPRIO_BLKDRVCANCLIENT¶

#define BLKDRVCANCLIENTKEY_INT_CYCLETIME “CycleTimeMs”¶

Cycle Time in milliseconds

#define BLKDRVCANCLIENTKEY_INT_CYCLETIME_DEFAULT 1¶

#define BLKDRVCANCLIENTKEY_INT_NUMCHANNELS “NumChannels”¶

Number of Blockdriver Instances. Default: 1

#define BLKDRVCANCLIENTKEY_INT_NUMCHANNELS_DEFAULT 1¶

#define BLKDRVCANCLIENTKEY_INT_NETID “NetId”¶

Network ID of the blockdriver instance Example: 0.NetId=0

#define BLKDRVCANCLIENTKEY_INT_NETID_DEFAULT 0¶

#define BLKDRVCANCLIENTKEY_INT_BAUDRATE “Baudrate”¶

Baudrate of the blockdriver instance Example: 0.Baudrate=1000

#define BLKDRVCANCLIENTKEY_INT_BAUDRATE_DEFAULT 1000¶

#define BLKDRVCANCLIENTKEY_INT_NODEID “NodeId”¶

NodeID of the blockdriver instance Example: 0.NodeId=1

#define BLKDRVCANCLIENTKEY_INT_SERVERID “ServerId”¶

Deprecated: NodeID of the associated BlkDrvCanServer. Use only for old BlkDrvCanServer (before 3.5 SP6). Example: 0.ServerId=10

#define BLKDRVCANCLIENTKEY_INT_SERVERID_DEFAULT 0¶

#define BLKDRVCANCLIENTKEY_INT_BLOCKTRANSFER “BlockTransfer”¶

Enables BlockTransfer. Default = TRUE Example: 0.BlockTransfer=1

#define BLKDRVCANCLIENTKEY_INT_BLOCKTRANSFER_DEFAULT 1¶

#define BLKDRVCANCLIENTKEY_STRING_INTERFACE_NAME “Name”¶

Overwrites the default network interface name. Default: BlockDriver Instance 0 ==> BlkDrvCanClient BlockDriver Instance 1 ==> BlkDrvCanClient1 BlockDriver Instance 2 ==> BlkDrvCanClient2 Example: 0.Name=CANBus
