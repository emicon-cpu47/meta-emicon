OpcUa_EncodeableType* OpcUaGetEncodableObjectType (OpcUa_UInt32 ui32TypeId)¶

Get the adress of an encodable object type description. This adress can be passed directly to OpcUaEncodableObjectCreate or OpcUaEncodableObjectCreateExtension. As ui32TypeId use the OpcUaId_... Macros. If the type is not available the function will return NULL.
