#define GWCLIENTKEY_INT_INACTIVITY_TIMEOUT “InactivityTimeoutMs”¶

Setting to modify the default value of the Gateway inactivity timeout in ms. This default value is only used, if the Gateway connect is started by BeginConnectToGateway() instead of BeginConnectToGateway2(). If no piece of data (also no keepalive) is received or sent wihtin this time, the GwClient closes the connection to the Gateway.
