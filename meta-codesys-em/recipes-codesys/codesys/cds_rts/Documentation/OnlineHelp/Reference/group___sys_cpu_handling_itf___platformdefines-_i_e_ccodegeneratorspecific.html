Macro Definition Documentation¶

#define SYSCPU_EXTLIBCALL_REGISTER_SAVE_AREA 0¶

Size of the register save area in number of saved registers! Is used for locating the return address of an external library call in SysCpuGetCallstackEntry2() .

#define SYSCPU_EXTLIBCALL_ADJUST_OFFSET (offset)¶

Processor specific adjustment of offset for stack examination. Is used for locating the return address of an external library call in SysCpuGetCallstackEntry2() .
