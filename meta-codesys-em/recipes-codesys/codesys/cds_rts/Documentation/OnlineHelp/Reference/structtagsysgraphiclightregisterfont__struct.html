Detailed Description¶

Function to register a font, which is downloaded by the user.

Font data

Parameters

[in] udiDataSize Font data size
[in] pszFontName Name of the font that should be created.
[in] iFontSize Size of the font.
[in] udiFontFlags The font flags.

Returns:

*RESULT: Returns the runtime system error code (see CmpErrors.library).
