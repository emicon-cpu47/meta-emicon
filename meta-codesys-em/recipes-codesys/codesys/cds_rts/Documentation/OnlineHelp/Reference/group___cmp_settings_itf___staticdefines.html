Macro Definition Documentation¶

#define STD_SETTINGS_DATABASE “CODESYSControl”¶

This define can be used to call always the standard settings database.

#define STD_SETTINGS_DATABASE_OLD2 “CoDeSysControl”¶

#define STD_SETTINGS_DATABASE_OLD “CoDeSysSP”¶

This define can be used to call always the standard settings database.

#define FILE_END_OF_LINE_DELIMITER “rn”¶

Delimiter for the end of line, if a settings file is used as backend.

#define CMPSETTINGS_NUM_OF_STATIC_FILES 1¶

Number of static setting files. Only used by CmpSettings component to handle configuration files in static data.

#define CMPSETTINGS_MASTER_CONFIG “Master.cfg”¶

Name of the master configuration file, if standard config file could not be opened

#define CMPSETTINGS_STD_CONFIG STD_SETTINGS_DATABASE ”.cfg”¶

Name of the standard configuration file

#define USERDB_OBJECT_SETTINGS “Device.Settings”¶

Predefined objects in the runtime

#define CMPSETTINGS_CACHED 0¶

Options of the parameter bCached
