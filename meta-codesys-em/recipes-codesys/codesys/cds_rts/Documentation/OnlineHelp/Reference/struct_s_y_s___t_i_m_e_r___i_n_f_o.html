Detailed Description¶

Timer information structure that contains all information for the SysTimerOS implementation to handle one sepcific timer object.

RTS_HANDLE SYS_TIMER_INFO::hParam¶

Handle to a parameter, that must be transmitted as a parameter for the callback routine! Optional event to wake up the callback routine (task). This event is delivered in SysTimerCreateEvent.

RTS_HANDLE SYS_TIMER_INFO::hSysTimer¶

Handle to the system timer object in the OS implementation

RTS_HANDLE SYS_TIMER_INFO::hTimerToReset¶

SysTimer handle to the next timer that mast be activated at the return code of the actual timer callback

RTS_SYSTIME SYS_TIMER_INFO::tIntervalNs¶

Interval of the timer in nanoseconds

RTS_SYSTIME SYS_TIMER_INFO::tStartTime¶

Start time of the timer event, when the callback is called

RTS_SYSTIME SYS_TIMER_INFO::tLastExecuteNs¶

Last execution time of the timer

unsigned long SYS_TIMER_INFO::ulType¶

Timer type. See corresponding category.

unsigned long SYS_TIMER_INFO::ulPriority¶

Piority of the timer object. Typically between [0..SYSTIMER_NUM_OF_TIMER_PRIOS - 1]: 0=highest prio, 255=lowest prio

SYS_TIMER_CALL_CONTEXT SYS_TIMER_INFO::tCallContext¶

Actual register context of the calling callback. Can be used to make a context switch.

SYS_TIMER_BP_CONTEXT SYS_TIMER_INFO::tBPContext¶

Actual register context of the actual breakpoint position

int SYS_TIMER_INFO::bIECFunction¶

Flag to mark, if callback function is an IEC function
