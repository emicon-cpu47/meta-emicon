Macro Definition Documentation¶

#define APPBP_FLAG_BEFORE_CODE_PATCH 0x00000001¶

Flags of code patch event Is called before patching the code

#define APPBP_FLAG_AFTER_CODE_PATCH 0x00000002¶

Is called after patching the code

#define APPBP_FLAG_CODE_PATCHED 0x00000004¶

The event handler mark this flag to specify, that the code is patched in the event handler! So no default patching is done.
