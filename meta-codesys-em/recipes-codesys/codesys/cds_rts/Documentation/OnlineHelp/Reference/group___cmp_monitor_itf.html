RTS_RESULT MonitoringHasFeature (unsigned long ulFeatures)¶

Routine to check, if a scheduler has the specified feature.

Parameters

[in] ulFeatures Feature flags, See corresponding category “Features”.

Returns:

ERR_OK if the flags are supported, an error code otherwise

RTS_RESULT MonitoringReadValue (APPLICATION *pappl, BINTAGREADER *preader, RTS_UI8 *pbyValue, RTS_UI16 usSize)¶

Routine to read a value via a monitoring service. Note: this function is equivalent to calling MonitoringReadValue2 with bIec = 0.

Parameters

[in] pappl Pointer to application.
[in] preader the reader specifying the location of the value to monitor.
[in] pbyValue pointer to the destination address were the retrieved value is written.
[in] usSize size of value to read.

Returns:

ERR_OK if the value could be retrieved successfully
