#define EVT_SysNativeControlCreate2 MAKE_EVENTID (EVTCLASS_INFO , 6)¶

This additional event is sent when the method SysNativeControlCreate is called. It returns instead of an operation system handle to a native control a pointer to a SysNativeControlCustom structure.
