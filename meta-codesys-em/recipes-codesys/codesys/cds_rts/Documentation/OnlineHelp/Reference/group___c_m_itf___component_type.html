Macro Definition Documentation¶

#define CMPTYPE_STANDARD UINT16_C(0x0100)¶

A component can only be a member of type CMPTYPE_STANDARD or type CMPTYPE_SAFETY. To separate the components in detail, we use the types CMPTYPE_STATIC, CMPTYPE_DYNAMIC, CMPTYPE_SYSTEM or CMPTYPE_IEC. Standard component category

#define CMPTYPE_SAFETY UINT16_C(0x0200)¶

Safety component category

#define CMPTYPE_ALL (CMPTYPE_STANDARD | CMPTYPE_SAFETY )¶

Can be used to select all components. Only a mask to select STANDARD and SAFETY.

#define CMPTYPE_STATIC UINT16_C(0x0001)¶

Static linked component

#define CMPTYPE_DYNAMIC UINT16_C(0x0002)¶

Dynamically linked component

#define CMPTYPE_SYSTEM UINT16_C(0x0004)¶

System component

#define CMPTYPE_IEC UINT16_C(0x0008)¶

IEC component (IEC code is registered as a component)

#define CMPTYPE_APP_CMODULE UINT16_C(0x0010)¶

Application C-Module component
